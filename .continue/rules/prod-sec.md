# Continue Rules: prod-sec

`prod-sec` is a defensive secure-code-review skill.

Before editing or using the repository:

1. Read `SKILL.md`.
2. Review only local repositories, configuration, dependencies, and secrets in scope.
3. Preserve responsible-use language.
4. Keep scripts local, bounded, and evidence-oriented.
5. Avoid exploit payloads, brute force, recon, unauthorized access, persistence, evasion, malware, credential theft, destructive actions, and real data exfiltration.
