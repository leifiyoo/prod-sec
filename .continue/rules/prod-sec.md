# Continue Rules: prod-sec

`prod-sec` is an authorized security auditing skill.

Before editing or using the repository:

1. Read `SKILL.md`.
2. Keep tests scoped and safe-mode-first.
3. Preserve responsible-use language.

5. Avoid unauthorized access, persistence, evasion, malware, credential theft, destructive actions, and real data exfiltration.
